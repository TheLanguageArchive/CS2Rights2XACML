csrights2xacml ${project.version}.${buildNumber}, is a java tool
for converting the access rights found in the corpusstructure database 
of the LAT archiving software stack, into Fedora Commons XACML rules.
It is a jar file called csrights2xacml.jar wrapped by a runnable script
called csrights2xacml.sh

To Build:
mvn clean install

To run:
csrights2xacml.sh -?
